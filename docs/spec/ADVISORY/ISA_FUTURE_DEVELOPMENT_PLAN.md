## Enduring Roadmap Principles
- Advisory correctness beats feature breadth.
- Scope discipline beats generic platform expansion.
- User value is created by actionable guidance, not by speculative architecture.
- GS1 stakeholder value outranks generic ESG feature expansion.

## Condensed Direction
- Build from authority backbone and evidence substrate into a decision core.
- Turn the decision core into stakeholder-ready outputs.
- Expose the results through a compliance cockpit, explanation surface, and change-intelligence feed.

## Anti-Goals Preserved
- No generic ESG reporting suite.
- No compliance certification product.
- No customer-data validation service.
- No untraceable AI outputs.
